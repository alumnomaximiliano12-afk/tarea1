#MAXIMILIANO LIMON RAMIREZ 
Se aprendió qué es Git y GitHub, para qué sirven y cómo se usan para guardar 
versiones de archivos. 
También se aprendió cómo crear un repositorio, agregar archivos, hacer un commit y 
subirlos a GitHub. 

Comandos usados  

Comando  
git init 
Crear un repositorio en la carpeta
git add . 
Agregar los archivos para guardarlos 
git commit -m "mensaje" 
Guardar los cambios 
git remote add origin <url>
Conectar el proyecto con GitHub 
git push -u origin main 
Dificultades y solución 
Tuve dificultades ya que los pasos no eran claros y quedaban dudas de como hacerlo.
